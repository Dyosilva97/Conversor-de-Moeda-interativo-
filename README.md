💱 Conversor de Moeda

Este projeto é um Conversor de Moeda desenvolvido durante as aulas de Programação Web.
O objetivo é permitir que o usuário selecione uma moeda de origem, uma moeda de destino e insira um valor para realizar a conversão de forma simples e prática.

🎯 Objetivos do Projeto

Praticar conceitos de HTML, CSS e JavaScript.

Trabalhar com integração de APIs para buscar taxas de câmbio em tempo real.

Desenvolver habilidades de front-end interativo.

⚙️ Funcionalidades

Seleção de moeda de origem e destino.

Entrada de valor para conversão.

Exibição do resultado convertido.

Interface simples e responsiva.
